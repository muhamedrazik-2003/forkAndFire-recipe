About This Project

Odin's Recipe began as a simple assignment from the Odin Project. The goal was to build a website with a homepage that displays a few recipes and allows users to navigate to individual recipe pages.

The first version, created on August 2024, was a basic layout — mostly plain text and images without much styling or structure.

Fast forward 10 months to June 2025, I've completely redesigned the site with a modern, clean layout, improved typography, spacing, and subtle animations to enhance the user experience.

Currently i have only redesigned the website

This project reflects my growth as a frontend developer — from building with raw HTML and basic structure to creating responsive layouts, designing clean UIs, and thinking like a user.
